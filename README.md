



## 📒 RotateItem, [무작위 아이템 설정]
> 마인크래프트 서버 플러그인

---

#### 📖 어떤 플러그인인가요?
X축 또는 Z축을 선택하여 해당 방향으로만 이동할 수 있도록 제한하는 플러그인으로 파티클을 사용하여 이동 가능 경로를 표시해줍니다.

#### 📄 명령어
```
* /설정 - 무작위로 바뀔 아이템 설정
* /시작 - 무작위 변경 시작
* /중지 - 변경 중지
* /랜덤 - 강제로 무작위 변경 실행
```
